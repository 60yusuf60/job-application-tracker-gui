# Job Application Tracker (GUI)

A simple Python GUI application to track job applications. Built with **Tkinter**, it allows users to add new job applications with **Name, Company, Position, and Status**, and view all applications in a list.

## Features

- Add job applications with **Name, Company, Position, Status**
- Save applications locally in `jobs.json`
- View all saved applications with the **List Applications** button
- GUI built using **Tkinter** and **ttk**
